MySQL Shutdown

By Prathamesh Majalkar • Aug 10, 2025

This is one of the most common errors in XAMPP. To overcome this issue, follow these simple steps:

Steps:
  • 1: Close XAMPP or stop all the running services.
  • 2: Open the XAMPP folder on your PC (usually located in your C:\ drive).
  • 3: Inside the XAMPP folder, open the mysql folder.
  • 4: Copy the data folder and paste it in the same location with the name data_old. This is a backup to ensure you don’t lose your data.
  • 5: Open the original data folder.
  • 6: Important — read fully before proceeding: Delete the following 4 folders:
    1. phpmyadmin
    2. mysql
    3. performance_schema
    4. test
    Then delete all the files in this folder except ibdata1.
  • 7: Go back to C:\xampp\mysql and open the backup folder.
  • 8: Copy all the contents from the backup folder except the ibdata1 file.
  • 9: Paste them into the data folder (where you deleted the files earlier).
  • 10: Start XAMPP again — it should now work properly.
